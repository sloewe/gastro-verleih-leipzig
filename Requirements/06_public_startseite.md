# Task 06: Öffentlicher Bereich - Startseite

## Ziel
Präsentation der Produktkategorien in einem ansprechenden Kachellayout.

## Anforderungen
- Entwicklung der Landingpage (`/`).
- Abfrage aller Kategorien aus der Datenbank.
- Implementierung eines Responsive Grid Layouts (Kacheln).
- Jede Kachel zeigt den Kategorienamen und ein Hintergrundbild.
- Verlinkung der Kacheln zur jeweiligen Produktliste der Kategorie.

## Akzeptanzkriterien
- Startseite ist unter `/` erreichbar.
- Layout passt sich verschiedenen Bildschirmgrößen an (Tailwind CSS).
- Klick auf eine Kachel leitet korrekt weiter.
