# Task 03: Privater Bereich - Kategorien-CRUD

## Ziel
Verwaltungsoberfläche für Produktkategorien.

## Anforderungen
- Erstellung einer Livewire-Komponente zur Auflistung aller Kategorien.
- Formular zum Erstellen und Bearbeiten von Kategorien (Name, Bild).
- Löschfunktion (mit Sicherheitsabfrage).
- Integration in das Admin-Layout (Navigation).
- Validierung der Eingaben.

## Akzeptanzkriterien
- Admin kann Kategorien erstellen, sehen, bearbeiten und löschen.
- Slug wird automatisch generiert oder validiert.
- Formular nutzt Flux UI Komponenten.
