# Task 11: Deployment-Vorbereitung & Finalisierung

## Ziel
Vorbereitung der Anwendung für den Live-Betrieb.

## Anforderungen
- Konfiguration der Umgebungsvariablen (.env für Produktion).
- Optimierung der Assets (Vite build).
- Durchführung aller Tests (PHPUnit).
- Code-Formatting mit Laravel Pint.
- Einrichtung eines Deployment-Prozesses (z.B. Laravel Cloud oder Github Actions).

## Akzeptanzkriterien
- Alle Tests sind grün.
- Die Anwendung läuft im Produktionsmodus (APP_DEBUG=false).
- Assets sind kompiliert und werden korrekt ausgeliefert.
